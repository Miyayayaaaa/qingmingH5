# 清明返乡拼车 H5 页面

一个清明节返乡拼车主题的移动端H5页面。

## 功能特点

- 🎋 清明节主题设计
- 💬 弹幕评论滚动效果
- 🔄 TAB切换功能（同行找拼友 / 自驾求租车）
- 📱 移动端适配
- 🎨 绿色清新风格

## 预览

打开 `index.html` 文件即可在浏览器中查看效果。

## 部署方式

### 方法1: Vercel（推荐）
1. 访问 https://vercel.com
2. 注册/登录账号
3. 点击 "New Project"
4. 拖拽整个项目文件夹到页面上
5. 等待部署完成，获得访问链接

### 方法2: Netlify
1. 访问 https://www.netlify.com
2. 注册/登录账号
3. 点击 "Add new site" → "Deploy manually"
4. 拖拽整个项目文件夹到页面上
5. 等待部署完成，获得访问链接

### 方法3: GitHub Pages
1. 上传到GitHub仓库
2. 在仓库Settings中启用GitHub Pages
3. 选择main分支部署
4. 等待几分钟即可访问
